# Isochrone

Call the Valhalla API to generate isochrones, which are geographic areas that represent reachable regions within specified time intervals from a given location.

This sample UDF accepts parameters for latitude (lat), longitude (lng), a travel costing model (costing), and a list of time intervals (time_steps). Within the function, the `get_isochrone` method from the utils module is called, passing the specified parameters to request and retrieve the isochrone data from the Valhalla API. The function then returns the resulting isochrone data, which can be used to visualize travel ranges for each time step.

```python
@fused.udf
def udf(
    lat: float=40.776732,
    lng: float=-73.873708,
    costing: str="auto", # costing options: auto, pedestrian, bicycle, truck, bus, motor_scooter
    time_steps=[1, 5, 10, 15, 20, 25, 30]
):
    from utils import get_isochrone

    return get_isochrone(lat, lng, costing=costing, time_steps=time_steps)

```
