# H3 to Polygon

```python
@fused.udf
def udf(h3_cell='8739aab95ffffff'):
    import geopandas as gpd
    import h3
    from shapely.geometry import Polygon
    polygon = Polygon(h3.cell_to_boundary(h3_cell))
    gdf = gpd.GeoDataFrame([{'geometry': polygon}])
    return gdf
```
