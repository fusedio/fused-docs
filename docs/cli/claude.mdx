# `fused claude`

Manage the Fused plugin for Claude Code. Requires the `claude` CLI to be installed.

```
fused claude [SUBCOMMAND] [OPTIONS]
```

## Subcommands

| Subcommand | Description |
|---|---|
| `plugin add` | Install the Fused plugin from the marketplace |
| `plugin update` | Update to the latest plugin version |
| `plugin remove` | Uninstall the plugin |
| `add-mcp CANVAS_REF` | Register a canvas MCP endpoint with Claude Code |

---

## `fused claude plugin add`

Register the Fused marketplace and install the `fused@fused-marketplace` plugin. This teaches Claude Code the Fused-specific formats it needs — canvas structure, UDF patterns, widget schemas, and CLI usage.

```bash
fused claude plugin add
```

---

## `fused claude plugin update`

Update the installed plugin to the latest version.

```bash
fused claude plugin update
```

---

## `fused claude plugin remove`

Remove the Fused plugin from Claude Code.

```bash
fused claude plugin remove
```

---

## `fused claude add-mcp`

Register a canvas as an MCP endpoint with Claude Code so AI agents can query its UDFs.

```
fused claude add-mcp CANVAS_REF [OPTIONS]
```

| Flag | Description |
|---|---|
| `--token` | Treat ref as a share token (`fc_…`) |
| `--team` | Use a team canvas |
| `--id` | Treat ref as canvas ID |
| `--create-session-token / --no-create-session-token` | Create a session token (default: on) |
| `--session-max-age TEXT` | Session token lifetime (default: `1h`) |

```bash
fused claude add-mcp my_canvas
fused claude add-mcp fc_yourtoken --token
```

See [AI Coding Tools](/guide/working-with-udfs/udf-best-practices/ai-coding-tools) for the full Claude Code setup workflow.
