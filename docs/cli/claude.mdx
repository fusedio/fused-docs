# `fused claude`

Manage Fused for Claude Code (via the `claude` CLI).

```
fused claude [SUBCOMMAND] [OPTIONS]
```

## Subcommands

| Subcommand | Description |
|---|---|
| `add-mcp` | Add a canvas MCP server to Claude Code. |
| `plugin` | Add, update, or remove the Fused Claude plugin. |

## `fused claude add-mcp`

Add a canvas MCP server to Claude Code.

```
fused claude add-mcp CANVAS_REF [OPTIONS]
```

**Options**

| Flag | Description |
|---|---|
| `--token` | Treat CANVAS_REF as a shared canvas token (fc_...). |
| `--team` | Treat CANVAS_REF as a team canvas name. |
| `--id` | Treat CANVAS_REF as a canvas ID. |
| `--create-session-token / --no-create-session-token` | Create a session token for team-scoped canvases when none is passed. (default: `True`) |
| `--session-max-age TEXT` | Lifetime for --create-session-token (e.g. 30m, 1h, 1d). (default: `1h`) |


## `fused claude plugin`

Add, update, or remove the Fused Claude plugin.

| Subcommand | Description |
|---|---|
| `add` | Register the marketplace and install fused@fused-marketplace. |
| `remove` | Remove the fused plugin. |
| `update` | Update fused@fused-marketplace. |

### `fused claude plugin add`

Register the marketplace and install fused@fused-marketplace.

```
fused claude plugin add
```

### `fused claude plugin remove`

Remove the fused plugin.

```
fused claude plugin remove
```

### `fused claude plugin update`

Update fused@fused-marketplace.

```
fused claude plugin update
```
