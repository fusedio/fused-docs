# `fused files`

Manage files stored in your Fused account. Useful for persisting datasets, model outputs, or intermediate results between UDF runs.

```
fused files [SUBCOMMAND] [OPTIONS]
```

## `fused files list`

```
fused files list PATH [OPTIONS]
```

| Flag | Description |
|---|---|
| `--details` | Show file size and metadata |
| `-r, --recursive` | List recursively |

```bash
fused files list fd://
fused files list fd://my/path --details
fused files list fd://my/path -r
```

---

## `fused files upload`

```
fused files upload LOCAL_PATH REMOTE_PATH [OPTIONS]
```

| Flag | Description |
|---|---|
| `-r, --recursive` | Upload a directory recursively |
| `--dry-run` | Preview what would be uploaded (requires `-r`) |
| `--timeout FLOAT` | Upload timeout in seconds |

```bash
fused files upload data.csv fd://data/data.csv
fused files upload ./data/ fd://data/ -r
fused files upload ./data/ fd://data/ -r --dry-run
```

---

## `fused files download`

```
fused files download PATH LOCAL_PATH [OPTIONS]
```

| Flag | Description |
|---|---|
| `-r, --recursive` | Download a directory recursively |
| `--dry-run` | Preview what would be downloaded (requires `-r`) |

```bash
fused files download fd://data/data.csv ./data.csv
fused files download fd://data/ ./data/ -r
```

---

## `fused files get`

Print file contents to stdout.

```bash
fused files get fd://data/config.json
```

---

## `fused files sign_url`

Return a signed URL for direct access to a file.

```bash
fused files sign_url fd://data/output.parquet
```

---

## `fused files delete`

```
fused files delete PATH [OPTIONS]
```

| Flag | Description |
|---|---|
| `--max-deletion-depth TEXT` | Max directory depth to delete; use `"unlimited"` to delete recursively |

```bash
fused files delete fd://data/old_file.csv
fused files delete fd://data/old_dir/ --max-deletion-depth unlimited
```
