# CLI Reference

The `fused` CLI lets you manage every part of Fused from your terminal — run UDFs, push and pull canvases, upload files, manage secrets, and connect third-party integrations.

## Installation

```bash
pip install --upgrade fused
```

To run without a global install:

```bash
uvx fused <command>
```

Or inside a project venv:

```bash
uv run fused <command>
```

## Authentication

```bash
fused login     # open browser OAuth flow, persist credentials
fused logout    # clear credentials
fused whoami    # show your user info
fused whoami --team   # show your team info
```

## Global flags

These flags apply to every command:

| Flag | Env var | Description |
|---|---|---|
| `--env TEXT` | `FUSED_ENV` | Override the Fused environment |
| `--format [json\|text]` | `FUSED_CLI_FORMAT` | Output format — use `json` for machine-readable output in scripts |

```bash
fused canvas list --format json
FUSED_CLI_FORMAT=json fused canvas list
```

:::note
The `fused` package also installs the Python SDK for use in scripts and notebooks. See [Python installation](/guide/advanced-setup/local-installation) for SDK-specific setup, authentication, and dependency options.
:::

## Commands

| Command | Description |
|---|---|
| [`fused canvas`](/cli/canvas) | Manage canvases — list, pull, push, share, serve as MCP |
| [`fused run`](/cli/run) | Run a UDF and print the result |
| [`fused files`](/cli/files) | Manage files stored in Fused |
| [`fused secrets`](/cli/secrets) | Manage kernel and user secrets |
| [`fused integrations`](/cli/integrations) | Manage third-party OAuth integrations |
| [`fused json-ui`](/cli/json-ui) | Inspect and debug JSON-UI widget configs |
| [`fused udf-schema`](/cli/udf-schema) | Print the API schema for a UDF |
| [`fused claude`](/cli/claude) | Manage the Fused plugin for Claude Code |
