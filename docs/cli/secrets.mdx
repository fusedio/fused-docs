# `fused secrets`

Manage kernel and user secrets.

```
fused secrets [SUBCOMMAND] [OPTIONS]
```

## Subcommands

| Subcommand | Description |
|---|---|
| `delete` | Delete a kernel secret by key. |
| `get` | Get a secret value by key. |
| `list` | List secret keys. |
| `set` | Set a kernel secret value. |

## `fused secrets delete`

Delete a kernel secret by key.

```
fused secrets delete KEY [OPTIONS]
```

**Options**

| Flag | Description |
|---|---|
| `--client-id TEXT` | Override kernel client ID when deleting kernel secrets. |
| `--user` | Rejected: user secrets are read-only. |


## `fused secrets get`

Get a secret value by key.

```
fused secrets get KEY [OPTIONS]
```

**Options**

| Flag | Description |
|---|---|
| `--user` | Read a user-owned secret instead of a kernel secret. |
| `--client-id TEXT` | Override kernel client ID when reading kernel secrets. |


## `fused secrets list`

List secret keys.

```
fused secrets list [OPTIONS]
```

**Options**

| Flag | Description |
|---|---|
| `--user` | List user-owned secrets instead of kernel secrets. |
| `--client-id TEXT` | Override kernel client ID when listing kernel secrets. |


## `fused secrets set`

Set a kernel secret value.

```
fused secrets set KEY VALUE [OPTIONS]
```

**Options**

| Flag | Description |
|---|---|
| `--client-id TEXT` | Override kernel client ID when writing kernel secrets. |
| `--user` | Rejected: user secrets are read-only. |
