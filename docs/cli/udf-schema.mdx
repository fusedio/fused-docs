# `fused udf-schema`

Print the API schema for a UDF.

```
fused udf-schema CANVAS [UDF] [OPTIONS]
```

## Options

| Flag | Description |
|---|---|
| `--stdin` | Read UDF source code from standard input (same as passing multiline code to fused.load). |
