# `fused udf`

Search and manage UDFs.

```
fused udf [SUBCOMMAND] [OPTIONS]
```

## Subcommands

| Subcommand | Description |
|---|---|
| `search` | Search UDFs by natural language query. |

## `fused udf search`

Search UDFs by natural language query.

```
fused udf search QUERY [OPTIONS]
```

**Options**

| Flag | Description |
|---|---|
| `--top-k INTEGER` | Number of results to return from each source. (default: `5`) |
| `--mode [all\|user\|public]` | Which UDF sources to search: user (personal), public (community), or all. (default: `all`) |
