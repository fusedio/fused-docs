# Data Ingestion

Fused supports ingesting various data types into your workflows.

**For geospatial data:** See the comprehensive [Geospatial Data Ingestion](/tutorials/Geospatial%20with%20Fused/geospatial-data-ingestion/) guide with detailed tutorials and examples.

**For non-geospatial data:** Use standard Python libraries (`pandas`, `requests`, etc.) within your UDFs to load CSV, JSON, APIs, and other data sources.


### `ingest_nongeospatial`

Ingest dataframes without a spatial component.

```python showLineNumbers
# Run this locally - not in Workbench
job_id = fused.ingest_nongeospatial(
    input=df,
    output="s3://sample-bucket/file.parquet",
).run_batch()
```
