# Data Ingestion

Fused supports ingesting various data types into your workflows.

:::note For geospatial data

See the comprehensive [Geospatial Data Ingestion](/tutorials/Geospatial%20with%20Fused/geospatial-data-ingestion/) guide with detailed tutorials and examples.
:::


### [`ingest_nongeospatial`](/python-sdk/top-level-functions/#fusedingest_nongeospatial)

Ingest dataframes without a spatial component using [`fused.ingest_nongeospatial`](/python-sdk/top-level-functions/#fusedingest_nongeospatial).

```python showLineNumbers
job_id = fused.ingest_nongeospatial(
    input=df,
    output="s3://sample-bucket/file.parquet",
).run_batch()
```
