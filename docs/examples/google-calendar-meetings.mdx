Knowing your next meeting starts in 30 minutes doesn't help if the commute takes 45. Calendar apps show times but ignore the real world — where you are, how long it takes to get between locations, and whether the gap between back-to-back meetings is actually enough. With Fused, you can pull your Google Calendar events, compute routing and travel times, and surface schedule conflicts to an AI agent.

## How It Works

### 1. Pull events from Google Calendar

A Fused UDF connects to the Google Calendar API and loads the day's events — meetings, blocks, and appointments — into a DataFrame with start times, end times, and locations.

### 2. Geocode and route between meetings

Downstream UDFs geocode each event's location, then call the Google Routes API to get driving time between consecutive meetings. The pipeline compares the gap between meetings against the actual travel time and marks each transition as **On Time** or **Will Be Late**.

### 3. Expose to an AI agent

The Canvas publishes the schedule analysis as an agent-callable endpoint. You can ask an AI agent like Claude practical questions:

- *"Will I make all my meetings on time today?"*
- *"Which meeting am I most likely to be late to?"*
- *"How much buffer do I have between my 2pm and 3pm?"*

### 4. Adjust and re-check

After editing your calendar in Google Calendar — moving a meeting, adding buffer time — you can re-run the Fused pipeline to verify the updated schedule works. The feedback loop is immediate.

## Build It Yourself

To run this with your own calendar you need:

**Google Maps API** — for geocoding and routing:
1. Get an API key from the [Google Cloud Console](https://console.cloud.google.com/apis/credentials)
2. Enable the [Routes API](https://console.cloud.google.com/apis/library/routes.googleapis.com) and [Calendar API](https://console.cloud.google.com/apis/library/calendar-json.googleapis.com)
:::info Store your API key in Fused Secrets
In Fused Workbench, go to **Preferences** → **Environment** → **Secrets management** ([docs](/workbench/preferences#secrets-management)) and add your API key with the name **`google_maps_api`**. The pipeline references it as `fused.secrets["google_maps_api"]`.
:::

**Google Calendar** — must be publicly accessible:
1. Open [Google Calendar](https://calendar.google.com) → click **⋮** next to your calendar → **Settings and sharing**
2. Under **Access permissions**, check **Make available to public**
3. Copy the **Calendar ID** (under *Integrate calendar*) and paste it into the `calendar_url` parameter of the `gcal_today_events` UDF in the Canvas
