- Create a new Default UDF, you'll have a demo housing dataset
- Ask the AI Assistant to make a simple interactive graph of the data:

```text
Make a simple interactive graph of this data
```

- Save (`Cmd + S` on MacOS or click the "Save" button)
- Click "URL" button to see deployed dashboard!

```bash
https://fused.io/.../run/file?
```
