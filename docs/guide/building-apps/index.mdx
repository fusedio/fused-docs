# Building Apps

Create interactive dashboards and apps powered by your UDFs.

## App Builder

Workbench includes a visual App Builder to create shareable apps:

1. Open [Workbench](https://www.fused.io/workbench)
2. Create or open a UDF
3. Click **App Builder**
4. Add widgets (sliders, dropdowns, text)
5. Share the app URL

## UDF as API

Every UDF becomes an HTTP endpoint:

```
https://fused.io/.../run/file?param=value
```

**Output formats:**

| Format | URL |
|--------|-----|
| JSON | `?format=json` |
| CSV | `?format=csv` |
| Parquet | `?format=parquet` |
| GeoJSON | `?format=geojson` |
| PNG | `?format=png` |

## Embed in Other Tools

### DuckDB

```sql
SELECT * FROM read_parquet('https://fused.io/.../run/file?format=parquet');
```

### Google Sheets

```
=IMPORTDATA("https://fused.io/.../run/file?format=csv")
```

### Notion

Use `/embed` block with your UDF URL.

### Python

```python
import pandas as pd
df = pd.read_parquet("https://fused.io/.../run/file?format=parquet")
```

## Interactive Parameters

Add parameters to your UDF for interactive controls:

```python
@fused.udf
def udf(
    start_date: str = "2024-01-01",
    metric: str = "count",
    threshold: float = 0.5
):
    ...
```

These become UI controls in App Builder and URL parameters for APIs.
