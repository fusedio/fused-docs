# Dynamic File API

Plug UDFs into other places as `.csv`, `.html`, `.json`, and more.

## Output Formats

UDFs can return data in various formats via the `format` query parameter:

### Vector Data Formats

| Format | Extension | Use Case |
|--------|-----------|----------|
| `parquet` | `.parquet` | Efficient columnar storage |
| `geojson` | `.geojson` | Standard GeoJSON |
| `json` | `.json` | JSON objects |
| `csv` | `.csv` | Comma-separated values |
| `mvt` | `.mvt` | Mapbox Vector Tiles |
| `feather` | `.feather` | Apache Arrow format |
| `excel` | `.xlsx` | Excel spreadsheet |
| `xml` | `.xml` | XML data |

### Raster Data Formats

| Format | Extension | Use Case |
|--------|-----------|----------|
| `png` | `.png` | Lossless images |
| `jpg`/`jpeg` | `.jpg` | Compressed images |
| `gif` | `.gif` | Animated images |
| `webp` | `.webp` | Modern image format |
| `tif`/`tiff` | `.tif` | GeoTIFF rasters |

## Example Usage

### Returning as JSON

```bash
https://www.fused.io/server/v1/realtime-shared/YOUR_TOKEN/run/file?format=json
```

### Returning as CSV

```bash
https://www.fused.io/server/v1/realtime-shared/YOUR_TOKEN/run/file?format=csv
```

### Returning as HTML

```bash
https://www.fused.io/server/v1/realtime-shared/YOUR_TOKEN/run/file?format=html
```

## Tile Endpoints

For map tile servers, use the `tiles` path with `{z}/{x}/{y}` template:

### Vector Tiles (MVT)

```bash
https://www.fused.io/server/v1/realtime-shared/YOUR_TOKEN/run/tiles/{z}/{x}/{y}?format=mvt
```

### Raster Tiles (PNG)

```bash
https://www.fused.io/server/v1/realtime-shared/YOUR_TOKEN/run/tiles/{z}/{x}/{y}?format=png
```

## Integrations

These dynamic endpoints work with:
- [DeckGL](/guide/data-input-outputs/export-api/geospatial#deckgl) - Interactive maps
- [Mapbox](/guide/data-input-outputs/export-api/geospatial#mapbox) - Web maps
- [QGIS](/guide/data-input-outputs/export-api/geospatial#qgis) - Desktop GIS
- [Felt](/guide/data-input-outputs/export-api/geospatial#felt) - Collaborative mapping
- Google Sheets - Spreadsheet import
