# Your First UDF

Create a UDF in under 2 minutes.

## Install

```bash
pip install fused
```

## Authenticate

```python
import fused
from fused.api import NotebookCredentials
credentials = NotebookCredentials()
```

Follow the link to authenticate.

## Create a UDF

```python
@fused.udf
def udf(name: str = "World"):
    return {"message": f"Hello, {name}!"}
```

## Run It

```python
result = fused.run(udf)
print(result)
# {'message': 'Hello, World!'}
```

## Run with Parameters

```python
result = fused.run(udf, name="Fused")
print(result)
# {'message': 'Hello, Fused!'}
```

## Return a DataFrame

```python
@fused.udf
def udf(rows: int = 10):
    import pandas as pd
    import numpy as np
    
    return pd.DataFrame({
        "x": np.random.randn(rows),
        "y": np.random.randn(rows),
        "label": [f"point_{i}" for i in range(rows)]
    })

result = fused.run(udf, rows=5)
print(result)
```

## Try in Workbench

1. Open [fused.io/workbench](https://www.fused.io/workbench)
2. Click **New UDF**
3. Paste your code
4. Click **Run**

Parameters become interactive sliders and inputs!
