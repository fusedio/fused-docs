# Example: Ingest GridMET

[GridMET](https://www.climatologylab.org/gridmet.html) is ~4 km gridded climate data (NetCDF). Ingest via: **NetCDF → Parquet** → **Parquet → hex** (per chunk) → **combine** into one hex Parquet. See [Vector to H3 — Example: GridMET](/guide/h3-analytics/vector-to-h3#example-gridmet-netcdfto-parquetto-hex).

## Read

Pre-ingested GridMET hex files (e.g. `s3://fused-asset/misc/hex/hex6_{year}_tmmx.parquet`) can be read by bounds with `common.read_hexfile`:

```python
common = fused.load("https://github.com/fusedio/udfs/tree/c576dc/public/common/")

@fused.udf
def udf(
    bounds: fused.types.Bounds = [-125.0, 24.0, -66.0, 49.0],
    hex_res: int = 6,
    year: int = 2024,
):
    tmmn_path = f"s3://fused-asset/misc/hex/hex6_{year}_tmmn.parquet"
    tmmx_path = f"s3://fused-asset/misc/hex/hex6_{year}_tmmx.parquet"
    df = common.read_hexfile(bounds, tmmn_path).rename(columns={"value": "tmmn"})
    df["tmmx"] = common.read_hexfile(bounds, tmmx_path)["value"]
    return df
```

Aggregate (min, max, mean, frost days, precipitation sum) per hex as needed. Use in [Joining](/guide/h3-analytics/joining#example-joining-env-variables) with CDL, soil, and irrigation.
