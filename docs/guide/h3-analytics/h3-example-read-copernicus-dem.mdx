# Example: Read Copernicus DEM

After [ingesting Copernicus DEM 90m](/guide/h3-analytics/raster-to-h3#example-copernicus-dem-90m) to H3, read by bounds and resolution for visualization or derivatives (e.g. slope).

## Read

```python
@fused.udf
def udf(
    bounds: fused.types.Bounds = [-125.82, 21.31, -65.63, 52.59],
    res: int = 5,
):
    path = "s3://fused-asset/hex/copernicus-dem-90m/"
    utils = fused.load("https://github.com/fusedio/udfs/tree/dd40354/community/joris/Read_H3_dataset/")
    df = utils.read_h3_dataset(path, bounds, res=res)
    common = fused.load("https://github.com/fusedio/udfs/tree/35afd2a/public/common/")
    con = common.duckdb_connect()
    df = con.execute("SELECT hex, data_avg AS elevation FROM df").df()
    return df
```

Returns `hex` and `elevation`. For slope/aspect, use [Aggregations — Derivative layers](/guide/h3-analytics/aggregations#derivative-layers) (e.g. `h3_grid_ring` + neighbor elevation). For styling, see [H3 Visualization](/guide/h3-analytics/visualization).
