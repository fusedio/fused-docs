# H3 Hexagons

H3 is a hierarchical spatial index using hexagonal cells. It's perfect for aggregating, joining, and analyzing geospatial data at scale.

## When to Use H3

- **Heatmaps** - Visualize density and counts
- **Aggregation** - Summarize data over areas
- **Multi-scale analysis** - Zoom in/out seamlessly
- **Joining datasets** - Combine data from different sources

## Quick Example

```python
import h3

# Point to H3
hex_id = h3.latlng_to_cell(37.7749, -122.4194, resolution=9)

# DataFrame to H3
df['hex'] = df.apply(
    lambda r: h3.latlng_to_cell(r['lat'], r['lon'], 9), 
    axis=1
)

# Aggregate
df.groupby('hex')['value'].sum()
```

## Resolution Quick Reference

| Resolution | Cell Size | Use Case |
|------------|-----------|----------|
| 4 | ~42 km | Continental |
| 7 | ~2.3 km | City-level |
| 9 | ~325 m | Neighborhood |
| 11 | ~46 m | Building-level |
| 13 | ~6.6 m | Precise |

See [Resolution Guide](/guide/h3-hexagons/h3-resolution-guide) for the full table.
