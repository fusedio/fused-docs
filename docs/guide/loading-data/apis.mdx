# APIs & STAC Catalogs

Fused UDFs can call any REST API or query STAC catalogs for satellite imagery.

---

## When to Use

- **REST APIs**: Real-time data, web services, third-party integrations
- **STAC Catalogs**: Satellite imagery, earth observation data

## STAC Catalogs

STAC (SpatioTemporal Asset Catalog) provides standardized access to satellite imagery.

| Catalog | Description | 
|---------|-------------|
| [AWS Earth Search](/reference/data-loading/specialized#earth-search-aws) | Sentinel, Landsat | 
| [Microsoft Planetary Computer](/reference/data-loading/specialized#planetary-computer) | Curated Earth data | 
| [Google Earth Engine](/reference/data-loading/specialized#google-earth-engine) | GEE catalog | 

---

## Quick Example

```python
@fused.udf
def udf(bounds: fused.types.Bounds = None):
    import pystac_client
    
    catalog = pystac_client.Client.open("https://earth-search.aws.element84.com/v1")
    items = catalog.search(collections=["sentinel-2-l2a"], bbox=bounds)
    # Process items...
```
