# Databases

Fused can connect to most databases. Store credentials securely with Fused Secrets, then query from any UDF.

---

## Setup

1. **Add credentials** in Workbench → Preferences → Secrets
2. **Access in UDFs** via `fused.secrets["KEY_NAME"]`

See [Secrets Management](/reference/workbench/preferences#secrets-management) for setup details.

---

## Supported Databases

| Database | Package | Use Case |
|----------|---------|----------|
| **Snowflake** | `snowflake-connector-python` | Data warehouse |
| **BigQuery** | `google-cloud-bigquery` | Google Cloud analytics |
| **PostgreSQL / PostGIS** | `sqlalchemy`, `geopandas` | Relational + geospatial |
| **DuckDB** | `duckdb` | In-memory analytics |
| **Redshift** | `redshift_connector` | AWS data warehouse |

---

## Quick Example

```python
@fused.udf
def udf():
    import snowflake.connector
    
    conn = snowflake.connector.connect(
        user=fused.secrets["SNOWFLAKE_USER"],
        password=fused.secrets["SNOWFLAKE_PASSWORD"],
        account=fused.secrets["SNOWFLAKE_ACCOUNT"],
    )
    # Query and return data
```
