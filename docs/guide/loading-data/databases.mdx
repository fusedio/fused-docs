# Databases

Fused can connect to most databases. Store credentials securely with Fused Secrets, then query from any UDF.

---

## Setup

1. **Add credentials** in Workbench → [Preferences](/reference/workbench/preferences) → [Secrets](/reference/workbench/preferences#secrets-management)
2. **Access in UDFs** via `fused.secrets["KEY_NAME"]`

See [Secrets Management](/reference/workbench/preferences#secrets-management) for setup details.

---

## Supported Databases

| Database                | Use Case                  |
|-------------------------|---------------------------|
| [**Snowflake**](/reference/data-loading/databases#snowflake)               | Data warehouse              |
| [**BigQuery**](/reference/data-loading/databases#bigquery)                 | Google Cloud analytics      |
| [**PostgreSQL / PostGIS**](/reference/data-loading/databases#postgresql--postgis)    | Relational + geospatial     |
| [**DuckDB**](/reference/data-loading/databases#duckdb-in-memory)                   | In-memory analytics         |
| [**Redshift**](/reference/data-loading/databases#redshift)                         | AWS data warehouse          |

## Quick Example

```python
@fused.udf
def udf():
    import snowflake.connector
    
    conn = snowflake.connector.connect(
        user=fused.secrets["SNOWFLAKE_USER"],
        password=fused.secrets["SNOWFLAKE_PASSWORD"],
        account=fused.secrets["SNOWFLAKE_ACCOUNT"],
    )
    # Query and return data
```
