# Google Earth Engine

Access the vast Google Earth Engine (GEE) data catalog directly from Fused UDFs.

---

## Why Use GEE with Fused?

- **Massive catalog**: Petabytes of satellite imagery and geospatial datasets
- **Server-side processing**: GEE processes data on Google's infrastructure
- **Combine with other sources**: Join GEE data with your own datasets in Fused

---

## Setup

1. Create a GEE service account
2. Add credentials to Fused Secrets
3. Initialize `ee` in your UDF

See [GEE Authentication](https://developers.google.com/earth-engine/guides/service_account) for service account setup.

---

## Example UDFs

Check out GEE examples in the **[UDF Catalog](https://www.fused.io/workbench#catalog)**:

- Search for "Earth Engine" or "GEE"
- Browse the `gee_bigquery` examples

## Quick Example

```python
@fused.udf
def udf(bounds: fused.types.Bounds = None):
    import ee
    
    ee.Initialize()
    
    # Access GEE data
    collection = ee.ImageCollection("COPERNICUS/S2_SR_HARMONIZED")
    # ...
```
