# Working with Local Files

Fused makes it easy to work with local files like CSV, Parquet, GeoJSON, and Shapefiles.

## Uploading Files

### Drag & Drop in Workbench

The easiest way to load a file is to **drag and drop** it directly into the Workbench File Explorer. Your file will be uploaded to your Fused storage and accessible from any UDF.

![Drag and drop files directly into Workbench](/img/tutorials/drag_and_drop.gif)

### Via Python

```python
# Upload a local file to Fused storage
fused.upload("local_file.csv", "fd://my-data/file.csv")
```

---

## Downloading / Exporting Data

Every UDF can export data in multiple formats.

### From the UI

In Canvas, click the **Download** button and select your format:
- CSV
- Parquet  
- GeoJSON
- Shapefile

### Via URL

Append `dtype_out_vector` to any UDF endpoint:

```
https://fused.io/fsh_.../run/file?dtype_out_vector=csv
https://fused.io/fsh_.../run/file?dtype_out_vector=parquet
https://fused.io/fsh_.../run/file?dtype_out_vector=geojson
```

## Supported Formats

| Format | Best For | Notes |
|--------|----------|-------|
| **Parquet** | Large datasets | Recommended - fast & compressed |
| **CSV** | Tabular data | Universal, but slower for large files |
| **GeoJSON** | Vector geospatial | Human-readable, good for web |
| **Shapefile** | Legacy GIS | Avoid for new projects |
| **GeoTIFF / COG** | Raster imagery | Use COG for cloud access |

See [File Formats Guide](/guide/loading-data/file-formats) for detailed recommendations.
