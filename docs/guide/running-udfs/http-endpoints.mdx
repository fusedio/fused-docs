# HTTP Endpoints

Every saved UDF can be called via HTTP, turning your data into an API.

---

## Endpoint Structure

```
https://www.fused.io/server/v1/realtime-shared/{token}/run/file?format=json
```

Get your token from Workbench → Save UDF → Share.

---

## Output Formats

Set `format` parameter based on your data type:

| Data Type | Formats |
|-----------|---------|
| **Tables** | `parquet`, `geojson`, `json`, `csv`, `mvt`, `excel` |
| **Images** | `png`, `jpg`, `tiff`, `webp` |
| **Text** | `json`, `html` |

```
.../run/file?format=geojson
.../run/file?format=png
```

---

## Passing Parameters

Add query parameters to the URL:

```python
# UDF definition
@fused.udf
def udf(city: str = "London", year: int = 2024):
    ...
```

```
.../run/file?format=json&city=Paris&year=2023
```

---

## Map Tiles

For tile servers, use the tiles endpoint:

```
.../run/tiles/{z}/{x}/{y}?format=png
```

This integrates with:
- [Mapbox](/reference/udf-patterns/integrations#mapbox)
- [QGIS](/reference/udf-patterns/integrations#qgis) (use `format=mvt` for vectors)
- [Felt](/reference/udf-patterns/integrations#felt)
- deck.gl / Leaflet

---

## Token Types

| Type | Use Case | Example |
|------|----------|---------|
| **Shared token** | Public access, no auth needed | `fsh_abc123...` |
| **Bearer token** | Authenticated, private UDFs | Account token |

:::warning
Never share your Bearer token. It grants full account access.
:::

---

## Caching

Endpoints cache responses when [caching](/guide/scaling-up/cache) is enabled. Same code + parameters = cached result.

---

## Example: Google Sheets

Fetch data directly into spreadsheets:

```
=IMPORTDATA("https://www.fused.io/.../run/file?format=csv")
```
