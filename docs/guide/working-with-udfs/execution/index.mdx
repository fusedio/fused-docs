# Execution

The 3 execution patterns in Fused.

- [Realtime](./realtime) - fused.run(), single execution, under 120s
- [Parallel](./parallel) - fused.submit(), multiple realtime at once
- [Batch Jobs](./batch-jobs) - Large async jobs, instance types
