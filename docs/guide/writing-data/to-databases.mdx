# Writing to Databases

Fused UDFs can push data directly to data warehouses and databases.

---

## How It Works

1. **Store credentials** in Fused Secrets
2. **Connect** using standard Python database libraries
3. **Write** using DataFrame methods or SQL

---

## Supported Databases

| Database | Write Method |
|----------|--------------|
| **Snowflake** | `write_pandas()` |
| **BigQuery** | `load_table_from_dataframe()` |
| **PostgreSQL** | `to_sql()` or `to_postgis()` |
| **Redshift** | `to_sql()` |

---

## Credentials Setup

Add database credentials in **Workbench → Preferences → Secrets**:

- `SNOWFLAKE_USER`, `SNOWFLAKE_PASSWORD`, `SNOWFLAKE_ACCOUNT`
- `POSTGRES_URL`
- etc.

See [Secrets Management](/reference/workbench/preferences#secrets-management).

---

## Append vs Replace

Most write methods support `if_exists` parameter:

```python
# Append to existing table
df.to_sql("table", engine, if_exists="append")

# Replace table
df.to_sql("table", engine, if_exists="replace")
```
