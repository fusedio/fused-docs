# API Reference

This is the API reference for the `fused` Python package, which allows you to build and run serverless data workflows.

The reference is intended to be limited to low-level descriptions of various programmatic functionality. If you're just getting started with Fused, we recommend looking at the [Guide](/guide/guide-overview) first or exploring an [Example](/examples/zonal-stats).

## Decorators

| Function | Description |
| --- | --- |
| [@fused.udf](/python-sdk/api-reference/fused-udf) | Decorator to define a User Defined Function |
| [@fused.cache](/python-sdk/api-reference/fused-cache) | Decorator to cache function results |

## Running UDFs

| Function | Description |
| --- | --- |
| [fused.run](/python-sdk/api-reference/fused-run) | Execute a UDF and return results |
| [fused.run_async](/python-sdk/api-reference/fused-run-async) | Execute a UDF asynchronously |
| [fused.submit](/python-sdk/api-reference/fused-submit) | Submit a UDF for parallel execution |
| [JobPool](/python-sdk/api-reference/jobpool) | Manage multiple concurrent job executions |

## Loading UDFs

| Function | Description |
| --- | --- |
| [fused.load](/python-sdk/api-reference/fused-load) | Load a UDF from file, URL, or Workbench |
| [fused.load_async](/python-sdk/api-reference/fused-load-async) | Load a UDF asynchronously |

## Data Access

| Function | Description |
| --- | --- |
| [fused.get_chunk_from_table](/python-sdk/api-reference/fused-get-chunk-from-table) | Read a chunk from a partitioned table |
| [fused.get_chunks_metadata](/python-sdk/api-reference/fused-get-chunks-metadata) | Get metadata for table chunks |
| [fused.find_dataset](/python-sdk/api-reference/fused-find-dataset) | Search for datasets in the catalog |
| [fused.register_dataset](/python-sdk/api-reference/fused-register-dataset) | Register a dataset in the catalog |

## File Operations

| Function | Description |
| --- | --- |
| [fused.download](/python-sdk/api-reference/fused-download) | Download files from URLs |
| [fused.file_path](/python-sdk/api-reference/fused-file-path) | Get local file path for cached files |
| [fused.ingest](/python-sdk/api-reference/fused-ingest) | Ingest geospatial data into optimized format |
| [fused.ingest_nongeospatial](/python-sdk/api-reference/fused-ingest-nongeospatial) | Ingest non-geospatial data |
