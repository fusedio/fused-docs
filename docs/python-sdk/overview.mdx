# API Reference

This is the API reference for the `fused` Python package, which allows you to build and run serverless data workflows.

The reference is intended to be limited to low-level descriptions of various programmatic functionality. If you're just getting started with Fused, we recommend looking at the [Guide](/guide/guide-overview) first or exploring an [Example](/examples/zonal-stats).

## Decorators

| Function | Description |
| --- | --- |
| [@fused.udf](/python-sdk/api-reference/udf-decorator) | Decorator to define a User Defined Function |
| [@fused.cache](/python-sdk/api-reference/cache) | Decorator to cache function results |

## UDF Execution

| Class/Function | Description |
| --- | --- |
| [Udf](/python-sdk/api-reference/udf) | UDF class — call directly or use `.map()` for parallel execution |
| [JobPool](/python-sdk/api-reference/jobpool) | Manage parallel job execution (returned by `udf.map()`) |
| [fused.run](/python-sdk/api-reference/run) | [Legacy] Execute a UDF |
| [fused.run_async](/python-sdk/api-reference/run-async) | [Legacy] Execute a UDF asynchronously |
| [fused.submit](/python-sdk/api-reference/submit) | [Legacy] Submit a UDF for parallel execution |

| Function | Description |

| [fused.load](/python-sdk/api-reference/load) | Load a UDF from file, URL, or Workbench |
