# Quickstart for Data Analysts

Explore data visually, build dashboards, and share insights.

## Get Started in Workbench

1. Go to [fused.io/workbench](https://www.fused.io/workbench)
2. Sign up or log in
3. Click **File Explorer** to browse data
4. Double-click a file to create a UDF

No installation required!

## Explore a Dataset

In the File Explorer, find a dataset and double-click to open it. Fused creates a UDF template:

```python
@fused.udf
def udf():
    import geopandas as gpd
    return gpd.read_file("s3://bucket/your_file.geojson")
```

Click **Run** to see it on the map.

## Filter and Transform

Add parameters to filter data interactively:

```python
@fused.udf
def udf(
    min_value: int = 0,
    max_value: int = 100,
    category: str = "all"
):
    import geopandas as gpd
    
    gdf = gpd.read_file("s3://bucket/data.geojson")
    
    # Filter by value range
    gdf = gdf[(gdf['value'] >= min_value) & (gdf['value'] <= max_value)]
    
    # Filter by category
    if category != "all":
        gdf = gdf[gdf['category'] == category]
    
    return gdf
```

Parameters appear as sliders and dropdowns in the UI!

## Build a Dashboard

1. Click **App Builder**
2. Add widgets (sliders, dropdowns, text)
3. Connect widgets to UDF parameters
4. Share the app URL

## Key Features

### Map Visualization

- Automatic map display for geospatial data
- Pan and zoom to explore
- Click features for details

### Data Preview

- See data tables alongside maps
- Download as CSV, GeoJSON, or Parquet

### Sharing

- Share UDF links with colleagues
- Embed maps in other tools
- Export to Google Sheets
