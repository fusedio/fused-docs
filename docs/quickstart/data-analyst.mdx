# Quickstart for Data Analysts

Explore data visually, build dashboards, and share insights.

## Get Started in Workbench

1. Go to [fused.io/workbench](https://www.fused.io/workbench)
2. Sign up or log in
3. Click **File Explorer** to browse data
4. Double-click a file to create a UDF

No installation required!

## Explore a Dataset

In the File Explorer, find a dataset and double-click to open it. Fused creates a template:

```python
@fused.udf
def udf():
    import pandas as pd
    return pd.read_parquet("s3://fused-sample/demo_data/housing_2024.parquet")
```

Click **Run** to see the data.

## Filter and Transform

Add parameters for interactive filtering:

```python
@fused.udf
def udf(
    min_bedrooms: int = 0
):
    import pandas as pd
    
    df = pd.read_parquet("s3://fused-sample/demo_data/housing_2024.parquet")
    
    # Filter by number of bedrooms
    if min_bedrooms > 0:
        df = df[df['bedrooms'] >= min_bedrooms]
    
    # Add derived column
    df['price_per_sqft'] = df['price'] / df['area']
    
    return df
```

Parameters appear as sliders and dropdowns in the UI!

## Download Results

In Canvas, click the "Share" button to get a link to your analysis:
```
https://udf.ai/[shared_token].html
```

You can then change the format to download the results in your preferred format:
```
https://udf.ai/[shared_token].csv      # CSV, perfect for Excel
https://udf.ai/[shared_token].parquet # Parquet, perfect for data tools
https://udf.ai/[shared_token].json    # JSON, perfect for APIs
https://udf.ai/[shared_token].html   # HTML, perfect for quick visualization or embedding
```

See more about [supported output formats](/guide/running-udfs/http-endpoints#output-formats).

## Export to Google Sheets

You can import your analysis directly into Google Sheets:

```
=IMPORTDATA("https://udf.ai/[shared_token].csv")
```
