# Quickstart for Data Analysts

Explore data visually, build dashboards, and share insights.

## Get Started in Workbench

1. Go to [fused.io/workbench](https://www.fused.io/workbench)
2. Sign up or log in
3. Click **File Explorer** to browse data
4. Double-click a file to create a UDF

No installation required!

## Explore a Dataset

In the File Explorer, find a dataset and double-click to open it. Fused creates a template:

```python
@fused.udf
def udf():
    import pandas as pd
    return pd.read_parquet("s3://fused-sample/demo_data/housing_2024.parquet")
```

Click **Run** to see the data.

## Filter and Transform

Add parameters for interactive filtering:

```python
@fused.udf
def udf(
    min_price: int = 100000,
    max_price: int = 1000000,
    city: str = "all"
):
    import pandas as pd
    
    df = pd.read_parquet("s3://fused-sample/demo_data/housing_2024.parquet")
    
    # Filter by price range
    df = df[(df['price'] >= min_price) & (df['price'] <= max_price)]
    
    # Filter by city
    if city != "all":
        df = df[df['city'] == city]
    
    # Add derived column
    df['price_per_sqft'] = df['price'] / df['area']
    
    return df
```

Parameters appear as sliders and dropdowns in the UI!

## Build a Dashboard

1. Click **App Builder**
2. Add widgets (sliders, dropdowns, charts)
3. Connect widgets to UDF parameters
4. Share the app URL

## Key Features

### Quick Aggregations

```python
@fused.udf
def udf(group_by: str = "city"):
    import pandas as pd
    
    df = pd.read_parquet("s3://bucket/sales.parquet")
    
    return df.groupby(group_by).agg({
        'revenue': 'sum',
        'orders': 'count',
        'price': 'mean'
    }).reset_index()
```

### DuckDB for SQL Queries

```python
@fused.udf
def udf(query: str = "SELECT * FROM data LIMIT 100"):
    import duckdb
    
    conn = duckdb.connect()
    return conn.execute(f"""
        WITH data AS (
            SELECT * FROM 's3://bucket/sales.parquet'
        )
        {query}
    """).df()
```

### Export to Google Sheets

Your UDF can be imported directly into Google Sheets:

```
=IMPORTDATA("https://fused.io/.../run/file?format=csv")
```

### Download Results

Export your analysis in multiple formats:
- CSV for Excel
- Parquet for data tools
- JSON for APIs
