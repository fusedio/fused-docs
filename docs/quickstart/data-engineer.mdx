# Quickstart for Data Engineers

Build APIs, run batch jobs, and create data pipelines.

## Setup

```bash
pip install fused
```

```python
import fused
from fused.api import NotebookCredentials
credentials = NotebookCredentials()
```

## Create an API Endpoint

```python
@fused.udf
def udf(table: str = "users", limit: int = 100):
    import duckdb
    
    conn = duckdb.connect()
    df = conn.execute(f"""
        SELECT * FROM 's3://bucket/{table}.parquet' 
        LIMIT {limit}
    """).df()
    
    return df
```

## Run Batch Jobs

```python
# Define processing UDF
@fused.udf
def process(file_path: str):
    import pandas as pd
    df = pd.read_parquet(file_path)
    # Process...
    return df

# Run over many files in parallel
files = ["s3://bucket/file1.parquet", "s3://bucket/file2.parquet", ...]
results = fused.submit(process, [{"file_path": f} for f in files])
```

## Ingest Large Datasets

```python
job = fused.ingest(
    input="s3://bucket/raw_data/",
    output="s3://bucket/processed/",
    partitioning_method="rows",
)
job.run_batch(instance_type="m5.4xlarge")
```

## Key Patterns

### Parameterized Endpoints

```python
@fused.udf
def udf(
    start_date: str = "2024-01-01",
    end_date: str = "2024-12-31",
    region: str = "US"
):
    # Query based on parameters
    ...
```

### Scheduled Runs

Call your UDF endpoint from any scheduler (Airflow, cron, etc.):

```bash
curl "https://fused.io/.../run/file?format=parquet" > output.parquet
```

### Output Formats

```
?format=parquet  # Parquet file
?format=csv      # CSV file
?format=json     # JSON
?format=geojson  # GeoJSON
```
