# Quickstart for Data Scientists

Build analysis workflows with Jupyter, run experiments, and share results.

## Setup

```bash
pip install "fused[all]"
```

```python
import fused
from fused.api import NotebookCredentials
credentials = NotebookCredentials()
```

## Your First Analysis UDF

```python
@fused.udf
def udf(threshold: float = 0.5):
    import pandas as pd
    
    # Load sample data
    df = pd.read_csv("s3://fused-sample/demo_data/housing_2024.csv")
    
    # Process
    df['price_per_sqft'] = df['price'] / df['area']
    df['above_threshold'] = df['price_per_sqft'] > threshold
    
    return df
```

## Run Locally

```python
result = fused.run(udf, engine="local")
print(result.head())
```

## Run Remotely

```python
result = fused.run(udf)  # Runs on Fused cloud
```

## Visualize in Workbench

2. Create a new UDF
3. Paste your code
4. See results in the data table

## Key Patterns

### Cache Expensive Operations

```python
@fused.cache
def load_model():
    import joblib
    return joblib.load("s3://bucket/model.pkl")

@fused.udf
def udf(data_path: str):
    model = load_model()
    df = pd.read_parquet(data_path)
    df['prediction'] = model.predict(df[features])
    return df
```

### Process in Parallel

```python
# Run same UDF over many parameter values
results = fused.submit(my_udf, [{"param": v} for v in range(100)])
```

### Experiment Tracking

```python
@fused.udf
def experiment(learning_rate: float = 0.01, epochs: int = 100):
    # Train model with parameters
    model = train(lr=learning_rate, epochs=epochs)
    return {"accuracy": model.accuracy, "loss": model.loss}

# Grid search
params = [{"learning_rate": lr, "epochs": e} 
          for lr in [0.01, 0.1] 
          for e in [50, 100]]
results = fused.submit(experiment, params)
```
