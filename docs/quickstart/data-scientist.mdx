# Quickstart for Data Scientists

Build analysis workflows with Jupyter, visualize on maps, and share results.

## Setup

```bash
pip install "fused[all]"
```

```python
import fused
from fused.api import NotebookCredentials
credentials = NotebookCredentials()
```

## Your First Analysis UDF

```python
@fused.udf
def udf(bounds: fused.types.Bounds = None):
    import geopandas as gpd
    
    # Load data
    gdf = gpd.read_file("s3://fused-sample/demo_data/subway_stations.geojson")
    
    # Filter to viewport
    if bounds:
        from shapely.geometry import box
        gdf = gdf[gdf.intersects(box(*bounds))]
    
    return gdf
```

## Run Locally

```python
result = fused.run(udf, engine="local")
print(result.head())
```

## Run Remotely

```python
result = fused.run(udf)  # Runs on Fused cloud
```

## Visualize in Workbench

1. Open [Workbench](https://www.fused.io/workbench)
2. Create a new UDF
3. Paste your code
4. See it on the map!

## Key Patterns

### Cache Expensive Operations

```python
@fused.cache
def load_data(path):
    return gpd.read_file(path)
```

### Process in Parallel

```python
results = fused.submit(udf, [{"param": v} for v in range(10)])
```

### Use H3 for Aggregation

```python
import h3
df['hex'] = df.apply(lambda r: h3.latlng_to_cell(r['lat'], r['lon'], 9), axis=1)
df.groupby('hex')['value'].sum()
```
