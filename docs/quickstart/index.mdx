## Universal Quick Start

### 1. Install

```bash
pip install fused
```

### 2. Authenticate

```python
import fused
from fused.api import NotebookCredentials
credentials = NotebookCredentials()
```

### 3. Create Your First UDF

```python
@fused.udf
def udf(name: str = "World"):
    return {"message": f"Hello, {name}!"}
```

### 4. Run It

```python
result = fused.run(udf)
print(result)
# {'message': 'Hello, World!'}
```

### 5. Try in Workbench

Open [Workbench](https://www.fused.io/workbench) and paste your UDF to see it run with a visual map.
