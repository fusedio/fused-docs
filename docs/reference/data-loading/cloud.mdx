# Cloud Storage Snippets

Copy-paste code for loading from cloud storage providers.

## Amazon S3

### Public Bucket

```python
import pandas as pd
df = pd.read_parquet("s3://fused-sample/demo_data/housing_2024.parquet")
```

### Private Bucket

```python
import pandas as pd
# Credentials loaded from Fused Secrets automatically
df = pd.read_parquet("s3://my-private-bucket/data.parquet")
```

### With boto3

```python
import boto3
import pandas as pd
from io import BytesIO

s3 = boto3.client('s3')
obj = s3.get_object(Bucket='bucket', Key='file.parquet')
df = pd.read_parquet(BytesIO(obj['Body'].read()))
```

## Google Cloud Storage

```python
import pandas as pd
df = pd.read_parquet("gs://bucket-name/file.parquet")
```

```python
import geopandas as gpd
gdf = gpd.read_file("gs://bucket-name/file.geojson")
```


## Fused Storage (`fd://`)

```python
import pandas as pd

username = fused.api.whoami()['handle']
df = pd.read_parquet(f"fd://{username}/my-data/file.parquet")
```
