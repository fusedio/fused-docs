# H3 Reference

Quick reference for H3 hexagon operations. For detailed guides, see [H3 Hexagons Guide](/guide/h3-hexagons/).

## Pages

- [Conversions](/reference/h3/conversions) - Convert between geometries and H3
- [Operations](/reference/h3/operations) - Aggregation, neighbors, resolution changes

## Most Used Snippets

### Point to H3

```python
import h3
hex_id = h3.latlng_to_cell(lat, lon, resolution=9)
```

### DataFrame to H3

```python
df['hex'] = df.apply(lambda r: h3.latlng_to_cell(r['lat'], r['lon'], 9), axis=1)
```

### Aggregate

```python
df.groupby('hex')['value'].sum()
```

### Change Resolution

```python
parent = h3.cell_to_parent(hex_id, res=6)  # Coarser
children = h3.cell_to_children(hex_id, res=10)  # Finer
```

### Read H3 Dataset

```python
hex_reader = fused.load("https://github.com/fusedio/udfs/tree/main/community/joris/Read_H3_dataset/")
df = hex_reader.read_h3_dataset(path, bounds, res=None)
```
