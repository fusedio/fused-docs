## Most Used

### Create a UDF

```python
@fused.udf
def udf(my_param: str = "default"):
    return {"result": my_param}
```

### Run a UDF

```python
result = fused.run(my_udf, my_param="value")
```

### Load Data

```python
import geopandas as gpd
gdf = gpd.read_file("s3://bucket/file.geojson")
```

### Cache a Function

```python
@fused.cache
def load_data(path):
    return pd.read_parquet(path)
```

### Point to H3

```python
import h3
hex_id = h3.latlng_to_cell(lat, lon, 9)
```
