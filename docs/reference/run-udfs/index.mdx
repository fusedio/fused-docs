# Running UDFs

There are 2 main ways to run UDFs:

1. [Realtime](/core-concepts/run-udfs/run-small-udfs) - the default way for quick execution with small resource requirements
2. [Batch Jobs](/core-concepts/run-udfs/run_large) - for longer execution time or higher resource requirements
