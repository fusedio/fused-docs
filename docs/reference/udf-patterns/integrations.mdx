# UDF Integrations

Integrate Fused UDFs with popular mapping and visualization tools via HTTP endpoints.

---

## DeckGL

[DeckGL](https://deck.gl/) is a highly performant framework for interactive map visualizations.

### Setup

1. Create a UDF and [generate an HTTPS endpoint](/guide/running-udfs/http-endpoints)
2. Create an `.html` file with DeckGL that loads data from your Fused endpoint

### H3HexagonLayer

```js
new H3HexagonLayer({
    id: "H3HexagonLayer",
    data: "https://www.fused.io/server/v1/realtime-shared/YOUR_TOKEN/run/file?format=json",
    extruded: true,
    getElevation: (d) => d.count,
    elevationScale: 20,
    getFillColor: (d) => [255, (1 - d.count / 500) * 255, 0],
    getHexagon: (d) => d.hex,
}),
```

### Vector Tile Layer

```js
new TileLayer({
  id: "VectorTileLayer",
  data: "https://www.fused.io/server/v1/realtime-shared/YOUR_UDF/run/tiles/{z}/{x}/{y}?format=geojson",
  maxZoom: 19,
  minZoom: 0,
  renderSubLayers: (props) => {
    const { boundingBox } = props.tile;
    return new GeoJsonLayer(props, {
      data: props.data,
      stroked: true,
      getLineColor: [0, 255, 10],
      getFillColor: [0, 40, 0, 0.5],
      bounds: [boundingBox[0][0], boundingBox[0][1], boundingBox[1][0], boundingBox[1][1]],
    });
  },
});
```

### Raster Tile Layer

```js
new TileLayer({
  id: "RasterTileLayer",
  data: "https://www.fused.io/server/v1/realtime-shared/YOUR_UDF/run/tiles/{z}/{x}/{y}?format=png",
  maxZoom: 19,
  minZoom: 0,
  renderSubLayers: (props) => {
    const { boundingBox } = props.tile;
    return new BitmapLayer(props, {
      image: props.data,
      bounds: [boundingBox[0][0], boundingBox[0][1], boundingBox[1][0], boundingBox[1][1]],
    });
  },
});
```

[Learn more about DeckGL](https://deck.gl/docs)

## Felt

[Felt](https://felt.com/) is a collaborative mapping platform for interactive maps.

### Raster Tiles

1. Generate a shared URL with `format=png` and `/{z}/{x}/{y}` template:

```
https://www.fused.io/server/v1/realtime-shared/YOUR_UDF/run/tiles/{z}/{x}/{y}?format=png
```

2. In Felt, click "Upload from URL" and paste the URL

### Vector Data

1. Generate a shared URL with `format=csv` or `format=parquet`:

```
https://www.fused.io/server/v1/realtime-shared/YOUR_UDF/run/file?format=csv
```

2. In Felt, click "Upload from URL" and paste the URL

[Learn more about Felt](https://felt.com/help)

## Kepler

[Kepler](https://kepler.gl/) is an open source tool for visualizing large geospatial datasets.

### Usage

1. Create a UDF that returns vector data
2. In the UDF Builder, click "Open in Kepler.gl" on the top-right menu
3. Wait for data transfer and click "Open in Kepler.gl" in the bottom-right

[Learn more about Kepler](https://kepler.gl/docs)

## Mapbox

[Mapbox GL JS](https://docs.mapbox.com/mapbox-gl-js/) creates interactive web maps.

### Vector Tiles

```html
<script>
map.on('load', () => {
    map.addSource('fused-source', {
        'type': 'vector',
        'tiles': [
            'https://www.fused.io/server/v1/realtime-shared/YOUR_UDF/run/tiles/{z}/{x}/{y}?format=mvt'
        ],
        'minzoom': 6,
        'maxzoom': 14
    });
    
    map.addLayer({
        'id': 'fused-layer',
        'type': 'line',
        'source': 'fused-source',
        'source-layer': 'udf', // Always 'udf' for Fused vector tiles
        'paint': {
            'line-color': 'rgb(53, 175, 109)',
            'line-width': 2
        }
    });
});
</script>
```

### Raster Tiles

```html
<script>
map.on('load', () => {
    map.addSource('fused-raster-source', {
        'type': 'raster',
        'tiles': [
            'https://www.fused.io/server/v1/realtime-shared/YOUR_UDF/run/tiles/{z}/{x}/{y}?format=png'
        ],
        'tileSize': 256
    });
    
    map.addLayer({
        'id': 'fused-raster-layer',
        'type': 'raster',
        'source': 'fused-raster-source'
    });
});
</script>
```

[Learn more about Mapbox GL JS](https://docs.mapbox.com/mapbox-gl-js/api/)

## QGIS

[QGIS](https://www.qgis.org/) is an open source desktop GIS platform.

### Raster Tiles

1. Generate a shared URL with `format=png`:

```
https://www.fused.io/server/v1/realtime-shared/YOUR_UDF/run/tiles/{z}/{x}/{y}?format=png
```

2. In QGIS: Right-click "XYZ Tiles" → "New Connection"
3. Paste the URL and configure the layer

### Vector Tiles

1. Generate a shared URL with `format=mvt`:

```
https://www.fused.io/server/v1/realtime-shared/YOUR_UDF/run/tiles/{z}/{x}/{y}?format=mvt
```

2. In QGIS: Right-click "Vector Tiles" → "New Connection"
3. Paste the URL and configure the layer

### Vector Files

1. Generate a shared URL with `format=geojson`:

```
https://www.fused.io/server/v1/realtime-shared/YOUR_UDF/run/file?format=geojson
```

2. In QGIS: Layer → Add Layer → Add Vector Layer
3. Paste the URL as the data source

[Learn more about QGIS](https://docs.qgis.org/)
