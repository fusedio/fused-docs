| Action | Shortcut |
|--------|----------|
| Run UDF | `Cmd/Ctrl + Enter` |
| Save UDF | `Cmd/Ctrl + S` |
| Toggle sidebar | `Cmd/Ctrl + B` |
| Search | `Cmd/Ctrl + P` |

### File Explorer

- Double-click file â†’ Creates template UDF

### UDF Builder

- **Tiled mode:** UDF reruns as you pan/zoom the map
- **Single mode:** UDF runs once for current viewport
- **Parameters:** Add to function signature, appear as UI controls

### Endpoints

After saving a UDF, get its HTTP endpoint:

```
https://fused.io/server/v1/realtime-shared/{token}/run/file
```

**Output formats:**
- `?format=parquet` - Parquet download
- `?format=csv` - CSV download
- `?format=geojson` - GeoJSON
- `?format=png` - PNG image
- `?format=mvt` - Vector tiles

### Storage Locations

| Prefix | Description |
|--------|-------------|
| `s3://bucket/` | AWS S3 |
| `gcs://bucket/` | Google Cloud Storage |
| `fd://path` | Fused-managed S3 |
| `/mnt/cache/` | Shared mount disk |
