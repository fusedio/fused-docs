# Analytics & Dashboard

### Interactive graphs for your data

- Create a new Default UDF, you'll have a demo housing dataset
- Ask the AI Assistant to make a simple interactive graph of the data:

```text
Make a simple interactive graph of this data
```

- Click Save
- Click "Share"
- Create a new Shared Token by clicking "Share" again
- Your endpoint is a now a fully deployed, live interactive graph!

```bash
https://fused.io/.../run/file?
```

Learn more about [creating a shared token](/core-concepts/run-udfs/run-small-udfs/#shared-token).
