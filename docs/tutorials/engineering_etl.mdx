# Engineering & ETL


### Bring Data to Fused


- Drag & Drop files into Fused
- fused.upload()
- Connect to your data sources (S3, GCS)
- fused.ingest() for large datasets
- def load_data + fused.cache to load small files


### Turn your data into an API

- Create a UDF to load data. 
- Return df / array / etc
- Shared token -> HTTPs endpoint


### Infrastructure

- Setup Github for your team
- Secrets
- On Prem option
