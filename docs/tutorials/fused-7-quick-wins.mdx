### Convert data quickly
```python
@fused.udf
def udf():
    import pandas as pd
    df = pd.read_excel("https://example.com/sales_data_2023.xlsx")
    df.to_parquet("s3://my-bucket/sales_data.parquet")
    
    return df
```

### Turn any data in API
```python
@fused.udf
def udf(path: str = "s3://fused-sample/demo_data/housing/housing_2024.csv"):
    import pandas as pd
    housing = pd.read_csv(path)
    housing['price_per_area'] = housing['price'] / housing['area']

    return housing['price_per_area'].mean().round(2)
```
