Load CSV files in a Fused UDF using pandas. You can upload your own CSV file to the provided [S3 bucket](/core-concepts/content-management/file-system/#fd-s3-bucket) through the [file browser](/workbench/file-explorer/) or use a public CSV file.


```python
@fused.udf
def udf():
    import pandas as pd

    # Read csv file
    # https://catalog.data.gov/dataset/supply-chain-greenhouse-gas-emission-factors-v1-2-by-naics-6
    df = pd.read_csv("https://pasteur.epa.gov/uploads/10.23719/1528686/SupplyChainGHGEmissionFactors_v1.2_NAICS_CO2e_USD2021.csv")
    return df
```
