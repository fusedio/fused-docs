[Geemap](https://geemap.org/) is a Python package to visually analyze geospatial data
with [Google Earth Engine (GEE)](https://earthengine.google.com/).

This snippet shows how to render a `GeoDataFrame` returned by a UDF in geemap, in a Jupyter Notebook. The workflow calls the [Overture Buildings UDF](https://github.com/fusedio/udfs/tree/main/public/Overture_Maps_Example) to load polygons, converts them to a GEE [FeatureCollection](https://developers.google.com/earth-engine/apidocs/ee-featurecollection), then renders them on the map as vectors.


## Authenticate Google Earth Engine

Running this command should initiate the authentication process.

```python showLineNumbers
import ee

ee.Authenticate()
ee.Initialize()
```




## Create a Raster Tile Layer

Load raster tiles from the [Cropland Data Layer (CDL) UDF](https://www.fused.io/workbench/catalog/CDLs_Tile_Example-a9f78d6d-4820-44ac-9032-5d19591e1a55) HTTP endpoint.

```python showLineNumbers
import geemap
from ipyleaflet import Map, TileLayer

# Create map
Map = geemap.Map(center=(37.8798, -121.2423), zoom=12, basemap="CartoDB.DarkMatter")

# Create Raster Tile
crop_type = "corn"
tiles = TileLayer(
    url=f"https://www.fused.io/server/v1/realtime-shared/UDF_CDLs_Tile_Example/run/tiles/{{z}}/{{x}}/{{y}}?dtype_out_raster=png&crop_type={crop_type}",
    name="CDLs Raster Tiles",
)
Map.add_layer(tiles)

Map

```

## Create a Vector Tile Layer

Load Vector Tiles from the Public [Overture Maps Example UDF](https://www.fused.io/workbench/catalog/Overture_Maps_Example-64071fb8-2c96-4015-adb9-596c3bac6787) HTTP endpoint.

```python showLineNumbers
import geemap
import requests
from ipyleaflet import Map, VectorTileLayer

# Create map
Map = geemap.Map(center=(51.5129,-0.1241), zoom=15, basemap="CartoDB.DarkMatter")

# Create Overture Buildings VectorTileLayer
vector_layer = VectorTileLayer(
    url="https://staging.fused.io/server/v1/realtime-shared/UDF_Overture_Maps_Example/run/tiles/{z}/{x}/{y}?dtype_out_vector=mvt",
    name="Overture Buildings Vector Tiles",
)
Map.add_layer(vector_layer)
```

## Create an EE Object Layer from a custom UDF

Define and run a custom UDF to buffer the geometry of a GeoDataFrame and display it as an EE Object GeoJSON layer. This will run in the local Python process by default and if the environment is [authenticated](/python-sdk/authentication/) it may run on the Fused server.

```python showLineNumbers
import geemap
from ipyleaflet import Map
import fused

# Create map
Map = geemap.Map(center=(38.8938,-77.0146), zoom=11)

# Define UDF
@fused.udf
def udf(url="https://www2.census.gov/geo/tiger/TIGER_RD18/STATE/11_DISTRICT_OF_COLUMBIA/11/tl_rd22_11_bg.zip"):
    import geopandas as gpd

    gdf = gpd.read_file(url)
    gdf.geometry = gdf.geometry.buffer(0.0001)
    return gdf

# Run UDF
gdf = fused.run(udf)

# Create EE Object GeoJSON layer
ee_object = geemap.geopandas_to_ee(gdf)
Map.add_layer(ee_object, {"color": "blue"}, "EE Object GeoJSON")

Map
```

## Create an EE Object Layer from a public UDF

Run the Public [Isochrone UDF](https://www.fused.io/workbench/catalog/Get_Isochrone-15dcecd8-c401-42a8-b9f0-a9dac3d7ddc3) and display the returned GeoDataFrame as an EE Object GeoJSON layer. Public UDFs run on the Fused server for free.

```python showLineNumbers
import geemap
from ipyleaflet import Map
import fused

# Create map
Map = geemap.Map(center=(40.8072,-73.8469), zoom=10)

# Run UDF
gdf = fused.run("UDF_Get_Isochrone",  lat=40.776732, lng=-73.73708).set_crs("WGS84")

# Create EE Object GeoJSON layer
ee_object = geemap.geopandas_to_ee(gdf)
Map.add_layer(ee_object, {"color": "red"}, "EE Object GeoJSON")

Map
```
