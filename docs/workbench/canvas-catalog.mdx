Canvases are now organized across multiple sections in the navigation menu and Home page.

You can explore and manage Canvases through:

- Home — quick access to Resources and Recent Canvases
- Recent — recently opened or edited Canvases
- Team — shared team Canvases
- Public — publicly available Canvases

Each section provides a catalog-style view where you can:

- Browse available Canvases
- Search and sort Canvases
- Open and continue working on existing Canvases
- Create new Canvases
- Delete Canvases (where permitted)

Selecting a Canvas opens it in the editor.
