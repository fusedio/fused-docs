Browse and quickly open files in S3 buckets. List file contents, preview files on a map, and create Fused UDFs from templates for common file types.




## Key features

The File explorer addresses common challenges associated with managing files in S3 buckets.

- **Browse S3 Buckets:** Easily navigate S3 buckets across organizations in a centralized UI.
- **Quick preview:** Explore data without writing code or downloading files.
- **Rapid Access:** Create custom UDFs for specific file types.
- **Flexible file types:** File type templates handle a wide range of file formats (audio, rasters, tables, and beyond) so you don't have to re-write custom openers.

## Core functionality

### Browse

Browse directories and files of S3 buckets. Use the search bar and filter options to find specific files quickly. Paste an S3 path to a publicly accessible bucket to explore your own bucket.
