Workbench is a browser-based platform to build with data. Write Python, visualize results instantly, and share your work.

## Views

**[Canvas](/workbench/canvas)** — Default view for building workflows. Connect UDFs together, create dashboards, and see results side-by-side.

**[UDF Builder](/workbench/udf-builder)** — Advanced view for geospatial workflows. Visualize multiple UDFs as map layers or inspect individual Table outputs.

## Tools

**[File Explorer](/workbench/file-explorer)** — Browse and open files directly from cloud storage (S3, GCS, Azure).

**[UDF Explorer](/workbench/udf-explorer)** — Organize and share your UDFs. Supports version control through GitHub.

Get started at [fused.io/workbench](https://www.fused.io/workbench).
