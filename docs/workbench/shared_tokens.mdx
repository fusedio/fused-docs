The Shared Tokens page lets you manage **canvas tokens** (and legacy per-UDF tokens) in one place.

Sharing is at the canvas level: when you [share a canvas](/workbench/udf-builder/canvas#share-modal), you get a canvas token (`fc_***`). All UDFs on that canvas are then callable by name at `udf.ai/fc_<token>/<udf_name>.<ext>`. Legacy per-UDF tokens (`fsh_***`) are still supported but deprecated.

You can view existing tokens, see their status, copy them, or delete them as needed.
