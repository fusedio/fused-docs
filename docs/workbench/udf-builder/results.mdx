Exploring and analyzing data involves scanning logs, previewing intermediary outputs, and debugging errors.

The Runtime section of the UDF Builder dynamically displays information related to UDF execution, including visual output, debug info, and request details available via the [toolbar](/workbench/udf-builder/runtime/#toolbar).


### Toolbar

The toolbar at the top of the Runtime pane includes a menu button for displaying [debug](/workbench/udf-builder/map/#debug) info and copying request details.
