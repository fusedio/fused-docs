Discover building blocks for your most important work.

The UDF Catalog offers a collection of portable and modular UDFs, making it easy to share and discover reusable code snippets within the Fused community and your team.


- Create a [new UDF](/workbench/udf-catalog/#new-udf)
- [Search](/workbench/udf-catalog/#search) the catalog across [UDF categories](/workbench/udf-catalog/#udf-categories)
- View the detailed [profile](/workbench/udf-catalog/#udf-profile) for each UDF
- [Add UDFs](/workbench/udf-catalog/#add-udf) to your UDF Builder

## New UDF

Create a new UDF by clicking the "New UDF" button. This opens a UDF with a default template.


## Search


Open the UDF Catalog by Clicking "Add UDF". Search UDFs by name, sort them, and toggle between gallery and list views. Click UDF cards to view their profiles or add them to the UDF Builder.

![Alt text](https://fused-magic.s3.us-west-2.amazonaws.com/docs_assets/image-20.png)





### UDF categories

- Public UDFs: Verified and available to everyone
- Community UDFs: Shared by the community and available to everyone
- Team UDFs: Shared within a team in a private [GitHub](/core-concepts/content-management/git/) repo
- Saved UDFs: A user's personal UDFs


## UDF profile

Select a UDF to view its profile. This page includes the UDF's `README.md`, last update date and author, link to GitHub source, and a code preview.

## Add UDF

Add UDFs to the UDF Builder by clicking the "Add UDF" button. This opens a UDF with a default template.
