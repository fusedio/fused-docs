## Committing a UDF

You can [publish your UDF to GitHub](/workbench/udf-explorer/#publish-a-udf-to-a-github-repository) directly from the Canvas.

1. Click **Versions** in the three-dot menu
2. (Optional) Add a commit message or change the target repository Select the UDF to commit
3. Select the UDF to commit
4. Click **Open PR**
5. Merge the PR on GitHub
