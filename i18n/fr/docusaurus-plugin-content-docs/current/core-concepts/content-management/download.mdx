# Télécharger

Téléchargez des fichiers distants vers le système local pour les rendre disponibles aux UDF à travers les exécutions. Les fichiers sont écrits sur un [disque](/core-concepts/content-management/file-system/#mntcache-disk) partagé entre tous les UDF dans une organisation.

## [`fused.download`](/python-sdk/top-level-functions/#fuseddownload)

Téléchargez n'importe quel fichier vers : `/mount/tmp/` auquel tout autre UDF peut ensuite accéder.


```python showLineNumbers
@fused.udf
def udf(url='https://www2.census.gov/geo/tiger/TIGER_RD18/STATE/11_DISTRICT_OF_COLUMBIA/11/tl_rd22_11_bg.zip'):
    out_path = fused.download(url=url, file_path='out.zip')
    print(out_path)
```

:::info
La fonction `download` définit un verrou pour s'assurer que le téléchargement se produit une seule fois, au cas où l'UDF serait appelé simultanément.
:::
