Fused fournit deux systèmes de fichiers pour rendre les fichiers accessibles à tous les UDF : un [bucket S3](/core-concepts/content-management/file-system/#fd-s3-bucket) et un [disque](/core-concepts/content-management/file-system/#mntcache-disk). L'accès est limité au niveau de l'organisation.

## `fd://` bucket S3

Fused provisionne un espace de noms de bucket S3 privé pour votre organisation. Il est idéal pour des ensembles de données à grande échelle, natifs du cloud ou accessibles globalement, tels que des tables ingérées, des GeoTIFFs et des fichiers qui doivent être lus en dehors de Fused.

Utilisez l'[Explorateur de fichiers](/workbench/file-explorer/) pour parcourir le bucket et voir son chemin complet.

![explorateur de fichiers](/img/workbench_s3file.png)

Les fonctions utilitaires de Fused peuvent y faire référence avec l'alias `fd://`.

```python showLineNumbers
job = fused.ingest(
    input="https://www2.census.gov/geo/tiger/TIGER_RD18/STATE/06_CALIFORNIA/06/tl_rd22_06_bg.zip",
    output="fd://census/ca_bg_2022/",
).run_batch()
```

## `/mnt/cache` disque

`/mnt/cache` est le chemin vers un disque monté pour stocker des fichiers partagés entre UDFs. C'est ici que `@fused.cache` et [`fused.download`](/core-concepts/content-management/download/) écrivent des données. Il est idéal pour les fichiers que les UDFs doivent lire avec une faible latence, les fichiers téléchargés, la sortie des fonctions mises en cache, les clés d'accès, `.env` et les poids des modèles ML.

Les UDFs peuvent interagir avec le disque comme avec un système de fichiers local. Par exemple, pour lister les fichiers dans le répertoire :

```python showLineNumbers
@fused.udf
def udf():
    import os
    for each in os.listdir('/mnt/cache/'):
        print(each)
```

### Dépannage

Si vous rencontrez l'erreur suivante, cela signifie que `/mnt/cache` n'est pas encore configuré pour votre environnement. Pour résoudre ce problème, veuillez contacter l'équipe de Fused pour l'activer.

```text
Error: No such file or directory: '/mnt/cache/'
```
