L'intégration GitHub vous permet de :
- Commiter votre code dans les dépôts Fused [public](https://github.com/fusedio/udfs/tree/main/public) & [communautaires](https://github.com/fusedio/udfs/tree/main/community) pour [que tout le monde puisse l'utiliser](https://www.fused.io/workbench/udf/catalog?tab=community) !
- Sauvegarde & Versioning des UDFs
- Collaboration entre équipes (pour les utilisateurs professionnels)

## Connexion de vos propres dépôts

<Tag color="#3399ff">Professionnel et Entreprise</Tag> _L'intégration de dépôts privés est accessible aux organisations disposant d'un abonnement Fused Professionnel & Entreprise (voir [Plans](https://www.fused.io/pricing))._

L'intégration Fused GitHub se branche directement sur vos propres dépôts.

### Configuration de l'intégration GitHub


1. Créez un nouveau [dépôt GitHub](https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-new-repository). Il n'y a pas de structure de dépôt imposée car Fused scanne l'ensemble du dépôt à la recherche d'UDFs, bien que le [dépôt UDFs public](https://github.com/fusedio/udfs) puisse servir de guide.

2. Installez l'application Fused GitHub pour votre organisation. Accédez à [cette URL GitHub](https://github.com/apps/fused-io), cliquez sur "Configurer", et sélectionnez l'organisation GitHub qui contient le dépôt.

![install fused github](/img/gh2.png)

3. Limitez l'application au dépôt cible. Il est recommandé de sélectionner uniquement le dépôt spécifique.

![install fused github 2](/img/gh3.png)


4. Une fois ce qui précède terminé, veuillez contacter l'équipe Fused pour finaliser la configuration (`info@fused.io`) :
- Fournissez à l'équipe Fused votre chemin et nom GitHub : `some-organization/some-repo`. (Exemple : `fusedio/udfs`)

5. Confirmez que l'intégration est activée en vérifiant que les UDFs du dépôt apparaissent sous l'onglet "UDFs d'équipe" dans le Catalogue UDF.

Vous pouvez voir tous les dépôts auxquels vous avez accès dans l'onglet Versions :

![Github Versions Tab](/img/core-concepts/github_versions_tab.png)
