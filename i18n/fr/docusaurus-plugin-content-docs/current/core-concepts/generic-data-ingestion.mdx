# Ingestion de données

Fused prend en charge l'ingestion de divers types de données dans vos workflows.

:::note Pour les données géospatiales

Consultez le guide complet sur l'[Ingestion de données géospatiales](/tutorials/Geospatial%20with%20Fused/geospatial-data-ingestion/) avec des tutoriels et des exemples détaillés.
:::


### [`ingest_nongeospatial`](/python-sdk/top-level-functions/#fusedingest_nongeospatial)

Ingestion de dataframes sans composant spatial en utilisant [`fused.ingest_nongeospatial`](/python-sdk/top-level-functions/#fusedingest_nongeospatial).

```python showLineNumbers
# Run this locally - not in Workbench
job_id = fused.ingest_nongeospatial(
    input=df,
    output="s3://sample-bucket/file.parquet",
).run_batch()
```
