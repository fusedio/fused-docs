# Exécution des UDF

Il existe 2 principales façons d'exécuter des UDF :

1. [Temps réel](/core-concepts/run-udfs/run-small-udfs) - la méthode par défaut pour une exécution rapide avec de faibles exigences en ressources
2. [Jobs par lots](/core-concepts/run-udfs/run_large) - pour un temps d'exécution plus long ou des exigences en ressources plus élevées
