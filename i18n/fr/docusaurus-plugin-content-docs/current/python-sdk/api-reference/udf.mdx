## Udf

La classe `Udf` est l'objet que vous obtenez lorsque vous définissez un UDF avec le
[`@fused.udf`](/python-sdk/top-level-functions/#fusedudf) décorateur, ou lorsque vous chargez
un UDF enregistré avec [`fused.load()`](/python-sdk/top-level-functions/#fusedload).

### to_fused

```python
to_fused(overwrite: bool | None = None, **kwargs: dict[str, Any])
```

Enregistrez cet UDF sur le service Fused.

**Paramètres :**

- **overwrite** (<code>bool | None</code>) – Si True, écrase l'UDF distant existant avec l'objet UDF.

---

### to_directory

```python
to_directory(where: str | Path | None = None, *, overwrite: bool = False)
```

Écrivez l'UDF sur le disque en tant que répertoire (dossier).

**Paramètres :**

- **where** (<code>str | Path | None</code>) – Un chemin vers un répertoire. Si non fourni, utilise le nom de la fonction UDF.

**Autres Paramètres :**

- **overwrite** (<code>[bool](#bool)</code>) – Si vrai, l'écrasement est autorisé.

---

### to_file

```python
to_file(where: str | Path | BinaryIO, *, overwrite: bool = False)
```

Écrivez l'UDF sur le disque ou l'objet de fichier spécifié.

L'UDF sera écrit sous forme de fichier Zip.

**Paramètres :**

- **where** (<code>str | Path | BinaryIO</code>) – Un chemin vers un fichier ou un objet de type fichier.

**Autres Paramètres :**

- **overwrite** (<code>[bool](#bool)</code>) – Si vrai, l'écrasement est autorisé.
