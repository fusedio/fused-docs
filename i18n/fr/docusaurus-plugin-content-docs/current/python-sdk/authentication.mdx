## Authentifiez-vous

Authentifiez le Fused [Python SDK](/python-sdk/) dans un Notebook Python.

Assurez-vous d'avoir le package `fused` [installé](/python-sdk/#python-install).

```python showLineNumbers
pip install "fused[all]"
```

Pour utiliser Fused, vous devez vous authentifier. Ce qui suit va stocker un fichier de credentials dans `~/.fused/credentials` :

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';


<Tabs className="unique-tabs">
  <TabItem value="notebook" label="Notebook" default>

    Dans une cellule de notebook, exécutez :

    ```py
    from fused.api import NotebookCredentials
    credentials = NotebookCredentials()
    ```

    Suivez l'URL dans votre navigateur pour vous authentifier.

  </TabItem>
  <TabItem value="script" label="Script Python">

    Exécutez ce qui suit : 

    ```python
    from fused.api import NotebookCredentials

    credentials = NotebookCredentials()
    print(credentials.url)
    ```

    Et suivez l'URL dans votre navigateur pour vous authentifier.

    :::info
    Lorsque vous exécutez du code depuis un environnement Python sans accès au navigateur, vous pouvez copier le fichier `~/.fused/credentials` à la même location dans le système cible.
    :::
  </TabItem>
</Tabs>

## Déconnexion

Déconnectez l'utilisateur actuel. Cela supprime les credentials enregistrés sur le disque et réinitialise l'API Fused globale.

```python showLineNumbers
import fused
fused.api.logout()
```

## Obtenir le token Bearer (d'accès)

Obtenez le token Bearer (parfois appelé token d'accès) du compte.

```python showLineNumbers
import fused
fused.api.access_token()
```

Cela peut être utile lors de l'appel de UDFs [via des requêtes HTTPS](/core-concepts/run-udfs/run-small-udfs/#private-token) en dehors du Fused Python SDK et de Workbench pour s'authentifier avec l'API Fused.

:::warning Ne partagez pas votre token Bearer

Ne partagez pas votre token Bearer avec qui que ce soit. Ceux-ci permettent d'usurper votre compte et doivent être traités comme tel.

:::
