## Python Install

**Version de Python :** 3.10+

```python
pip install "fused[all]"
```

<details>
<summary>Détails de l'installation</summary>

:::note
    L'installation de `fused` est requise si vous exécutez `fused` de votre côté (localement ou dans un environnement de développement). Si vous travaillez dans [Workbench UDF Builder](/workbench/udf-builder/) ou [App Builder](/workbench/app-builder/), `fused` est déjà installé pour vous.
:::

1. Configurez un environnement Python :

Nous utilisons `venv`, mais vous pouvez utiliser `conda` ou tout autre gestionnaire d'environnement en Python.

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Installez le package `fused` :

Vous pouvez uniquement installer le package de base, bien que nous recommandions d'ajouter les dépendances optionnelles :
```bash
pip install fused
```

Ou installez avec des dépendances optionnelles :
```bash
# Pour le traitement des données raster
pip install "fused[raster]"

# Pour le traitement des données vectorielles
pip install "fused[vector]"

# Installer toutes les dépendances optionnelles
pip install "fused[all]"
```

</details>



### Authenticate

La première fois que vous utilisez Fused, vous devrez [vous authentifier](/python-sdk/authentication/).

```python
from fused.api import NotebookCredentials

credentials = NotebookCredentials()
print(credentials.url)
```

Suivez l'URL dans votre navigateur pour vous authentifier.

## Basic API usage

Quelques exemples de base sur la façon d'utiliser `fused` pour exécuter des UDF :

#### Hello World UDF

```python
import fused

@fused.udf
def udf(x: int = 1):
    return f"Hello world {x + x}"

fused.run(udf)
```

```bash
>> Hello world 2
```

#### Simple data UDF

```python
import fused

@fused.udf
def udf(x: int = 3):
    import pandas as pd
    return pd.DataFrame({"x": range(x)})

fused.run(udf)
```

```bash
>>   | x |
|---|---|
| 0 | 0 |
| 1 | 1 |
| 2 | 2 |
```
