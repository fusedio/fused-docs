### Connectez-vous à Workbench

Commencez par vous connecter à [Workbench](https://www.fused.io/workbench), l'IDE web de Fused.

### Écrivez votre première Fonction Définie par l'Utilisateur (UDF)

Fused est construit autour des [Fonctions Définies par l'Utilisateur (UDFs)](/core-concepts/why/), des fonctions Python sans serveur qui ne nécessitent aucune configuration pour s'exécuter. Écrivez votre première UDF dans Workbench : 

```python
@fused.udf
def udf(path = "s3://fused-sample/demo_data/housing/housing_2024.csv"):
	import pandas as pd
	return pd.read_csv(path)
```

Vous avez maintenant chargé un DataFrame dans Fused ! 

Nous avons chargé des données sur le logement basées sur le [Kaggle Housing Prices Dataset](https://www.kaggle.com/datasets/yasserh/housing-prices-dataset). 

Pour mieux comparer les prix des maisons, nous pouvons calculer le prix par surface :

```python
@fused.udf
def udf(path: str = "s3://fused-sample/demo_data/housing/housing_2024.csv"):
    import pandas as pd
    housing = pd.read_csv(path)
    housing['price_per_area'] = round(housing['price'] / housing['area'], 2)
    
    return housing[['price', 'price_per_area']]
```

Vous avez réalisé votre première analyse !

### Vibe Coding un tableau de bord

Vous pouvez demander à l'Assistant IA de construire un tableau de bord interactif pour vous ! Une fois que vous êtes satisfait des données que vous avez dans votre analyse, tapez simplement :

```text
Build an interactive dashboard for me showing the price per area of the houses
```

L'IA ajoutera du code HTML pour rendre votre analyse dans un tableau de bord interactif. Enregistrez votre UDF et la version déployée de votre UDF sera mise à jour avec un tableau de bord interactif !

### Déployez votre tableau de bord

Dans Workbench :
1. Enregistrez votre UDF (`Cmd + S` sur MacOS ou cliquez sur le bouton "Enregistrer")
2. Cliquez sur le bouton "URL" pour voir le tableau de bord déployé !

Vous pouvez maintenant partager votre analyse avec votre équipe ou l'utiliser dans n'importe quelle application ! Tous les changements que vous apportez à votre UDF seront reflétés dans votre tableau de bord après que vous ayez actualisé !
