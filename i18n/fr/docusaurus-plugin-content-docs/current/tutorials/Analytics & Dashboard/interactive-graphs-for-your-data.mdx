- Créez un nouveau UDF par défaut, vous aurez un ensemble de données de logement de démonstration
- Demandez à l'Assistant AI de créer un graphique interactif simple des données :

```text
Make a simple interactive graph of this data
```

- Enregistrez (`Cmd + S` sur MacOS ou cliquez sur le bouton "Enregistrer")
- Cliquez sur le bouton "URL" pour voir le tableau de bord déployé !

```bash
https://fused.io/.../run/file?
```
