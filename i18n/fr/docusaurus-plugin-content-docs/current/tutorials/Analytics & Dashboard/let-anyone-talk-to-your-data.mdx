# Laissez quiconque parler à vos données

Rendez vos données disponibles via des serveurs MCP en 1 minute. Laissez quiconque parler à vos données directement en anglais simple ; aucune configuration requise !

### Essayez-le : [udf.ai](https://udf.ai/)

Sur [`udf.ai`](https://udf.ai/), vous pouvez :
- Poser directement des questions aux sources de données que nous avons connectées
- Connecter vos propres sources de données via des serveurs MCP

Cela rend vos données ou analyses disponibles pour _quiconque_ pour parler directement ; aucune configuration supplémentaire requise.

### Transformez vos données en un serveur MCP

Passez n'importe quelle URL qui renvoie du JSON à [`udf.ai`](https://udf.ai/). L'approche la plus simple : transformez vos données en un serveur MCP en utilisant un [Fused UDF](/core-concepts/write/).

Tout UDF peut être facilement transformé [en une API](/tutorials/engineering-etl/#turn-your-data-into-an-api) :

**Expérimental : Utiliser le profil de créateur MCP dans [AI Assistant](/workbench/ai-assistant/)**

1. Allez dans les [Préférences](/workbench/preferences/)
2. Activez "Profil AI MCP" 

![AI Profile Flag in Preferences](/img/tutorials/ai_profile_flag.png)

Cela vous donnera un nouveau profil appelé "Créateur MCP" dans l'AI Assistant :

![MCP Creator Profile in AI Assistant](/img/tutorials/MCP_creator_profile.png)

3. Passez à ce nouveau profil "Créateur MCP" dans l'AI Assistant

4. Demandez-lui de convertir le UDF actuel que vous avez ouvert en un serveur MCP : 

```text
Transformez ce UDF en un serveur MCP
```

L'IA devrait vous renvoyer un objet JSON avec les détails de votre serveur MCP. Par exemple :

```json
{ 
    "climate_parquet": { 
        "link": "https://www.fused.io/server/v1/realtime-shared/<your_udf_shared_token>/run/file?format=png,json", 
        "description": "Ce UDF effectue un traitement super cool. Appelez-le lorsque l'utilisateur veut faire quelque chose de super cool. Aucun paramètre n'est nécessaire.", 
        "parameters": {} 
    } 
}
```

5. Ajoutez ce JSON directement dans le JSON de configuration MCP de [udf.ai](https://udf.ai/)
6. Rechargez le chat

Posez toutes les questions que vous voulez à vos données sur [`udf.ai`](https://udf.ai/) !

```
Quelle a été la température la plus élevée en août 2024 ?
```

### Partage

Le JSON de configuration MCP est stocké dans le stockage local de votre navigateur. Vous ne pouvez pas encore partager des liens directement, mais vous pouvez copier votre JSON et l'envoyer à d'autres.

Puisque les points de terminaison UDF sont accessibles par quiconque ayant le lien par défaut, quiconque peut parler directement à vos données avec le bon JSON !
