# Discover data

### Explore data with Fused:

- Parcourez les buckets sans écrire de code dans [File Explorer](https://www.fused.io/workbench/files)
- Demandez à l'AI Assistant d'écrire un lecteur simple pour un fichier si vous ne savez pas comment faire :

```text
Use DuckDB to open this file and return first few rows: s3://fused-sample/demo_data/housing_2024.parquet
```

- Retournez les données sous forme de `pandas.DataFrame` pour voir un aperçu des données directement dans Workbench

Exemples :
- Explorez les données de démonstration Fused dans [File Explorer](https://www.fused.io/workbench/files?path=s3%3A%2F%2Ffused-sample%2Fdemo_data%2F) : `s3://fused-sample/demo_data/`


### Get a sample of data locally

Vous avez un grand ensemble de données dans le cloud, mais vous souhaitez en explorer une partie localement sur votre propre machine ? Écrivez une UDF simple pour retourner un sous-ensemble et appelez-la localement

(Nous supposons que vous avez [installé `fused`](/python-sdk/#python-install) & [authentifié](/python-sdk/authentication/#authenticate))

```python
import fused

@fused.udf
def udf(path: str='s3://fused-asset/data/era5/t2m/datestr=2024-01-01/0.parquet'):
    import pandas as pd
    df = pd.read_parquet(path)

    return df.head(10)
```

```python
sample = fused.run(udf)
```
