# Vibe Coded Timeseries

_Ceci est une version plus courte du cas d'utilisation [Climate Dashboard](/tutorials/Geospatial%20with%20Fused/use-cases/Climate%20Dashboard/), mais en codant les vibrations pour créer un tableau de bord de séries temporelles !_

Nous avons déjà ingéré 20 ans de données météorologiques ERA5, [disponibles sur Fused](s3://fused-sample/demo_data/ERA5/climate_data/) : `s3://fused-sample/demo_data/ERA5/climate_data/`

### Lire un fichier unique

En prenant le premier fichier, nous pouvons simplement demander à l'Assistant IA de le lire :

```
Read this file with DuckDB and return the first 10 rows: s3://fused-sample/demo_data/ERA5/climate_data/2005-01.pq
```

- Enregistrer (`Cmd + S` sur MacOS ou cliquez sur le bouton "Enregistrer")
- Cliquez sur le bouton "URL" pour voir le tableau de bord déployé !

Chaque fois que vous effectuez une mise à jour, votre graphique se mettra automatiquement à jour !
