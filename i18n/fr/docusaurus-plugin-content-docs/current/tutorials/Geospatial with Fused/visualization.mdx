## En dehors de Workbench

### Créez un serveur de tuiles de carte en quelques secondes

Cet exemple fonctionne mieux dans [Workbench](https://www.fused.io/workbench)

```python
@fused.udf
def udf(
   bounds: fused.types.Bounds,
   path: str="s3://fused-asset/data/tiger/state/tl_rd22_us_state 1pct.parquet"
):
   import geopandas as gpd
   
   df = gpd.read_parquet(path)
   df = df.cx[bounds[0]:bounds[2], bounds[1]:bounds[3]]
   df['area'] = df['geometry'].area.round(2)
   return df
```

- Passez à l'utilisation d'un [UDF de tuile](/tutorials/Geospatial%20with%20Fused/filetile/#tile)
- Enregistrez votre UDF
- Copiez le point de terminaison HTTPS

Modifiez les coordonnées de la tuile pour utiliser `{z}/{x}/{y}` :
```
https://.../run/tiles/{z}/{x}/{y}?format=parquet
```

Connectez ceci n'importe où pour livrer un serveur de tuiles de carte !
