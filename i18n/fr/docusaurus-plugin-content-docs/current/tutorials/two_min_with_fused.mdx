# 2 min avec Fused

Façons rapides de commencer avec Fused dans [Workbench](https://www.fused.io/workbench).

### Ouvrir un ensemble de données simple

```python
@fused.udf
def udf(path = "s3://fused-sample/demo_data/housing/housing_2024.csv"):
    import pandas as pd
    housing = pd.read_csv(path)
```

### Convertir des données rapidement

```python
@fused.udf
def udf():
    import pandas as pd
    df = pd.read_excel("s3://fused-sample/demo_data/housing_2024.xlsx")
    df.to_parquet("s3://fused-sample/demo_data/housing_2024.parquet")
    
    return df
```

### Retourner du code HTML & créer un graphique !

Vous pouvez retourner du code HTML directement dans Fused et transformer vos données en un graphique intégré :

```python
@fused.udf
def udf(path = "s3://fused-sample/demo_data/housing/housing_2024.csv"):
    from fastapi import Response
    import pandas as pd
    import altair as alt
    
    housing = pd.read_csv(path)
    housing['price_per_area'] = round(housing['price'] / housing['area'], 2)
    
    chart_html = alt.Chart(housing).mark_point().encode(
        x='price',
        y='price_per_area'
    ).to_html()

    
    return Response(chart_html.encode('utf-8'), media_type="text/html")
```
