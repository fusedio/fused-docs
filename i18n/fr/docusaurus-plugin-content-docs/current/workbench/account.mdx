Sur la page Compte, vous pouvez modifier votre nom d'utilisateur, consulter les informations de base de votre compte et vérifier votre utilisation dans la section Utilisation. Vous pouvez également accéder aux onglets Utilisateur, Env, Auth et Instance pour voir les paramètres d'authentification et les détails de l'instance.
