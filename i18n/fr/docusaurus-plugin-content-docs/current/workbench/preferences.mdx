Vous pouvez accéder à la page Préférences depuis Workbench en cliquant sur Préférences dans le coin inférieur gauche.

Cette page contient des préférences utilisateur générales, des fonctionnalités expérimentales, la sélection de thème, des paramètres de données et d'autres options de personnalisation.

![Emplacement du bouton Préférences](/img/workbench/preferences/new_preferences.png)

Cette page est grossièrement divisée en 2 sections :

![sections de la page préférences](/img/workbench/preferences/top_level_preference.png)

- **Environnement** : Configuration des [kernels](/workbench/preferences/#kernel), de la [gestion des secrets](/workbench/preferences/#secrets-management) et des [variables d'environnement](/workbench/preferences/#environment-variables). Voir ci-dessous pour plus de détails.
- **Préférences générales** : Telles que le thème, les formats de fichiers, etc. Celles-ci sont également recherchables depuis la [Palette de commandes](/core-concepts/best-practices/workbench-best-practices/#using-keyboard-shortcuts-command-palette).

## Kernel

Sélectionner un kernel dans Workbench définit l'environnement pour votre session, fournissant l'accès aux bibliothèques Python à l'exécution. Contactez l'équipe Fused via Slack, [Discord](https://discord.com/invite/BxS5wMzdRk) ou par email pour configurer de nouveaux kernels.

![kernel](/img/workbench/preferences/kernel.png)

## Gestion des secrets

Stockez et gérez des secrets qui sont sécurisés et chiffrés dans le backend.

![gestion des secrets](/img/workbench/preferences/secrets_management.png)

Ajoutez des secrets directement dans l'interface utilisateur en cliquant sur "+ Ajouter un nouveau secret"

Vous pouvez ensuite récupérer des secrets depuis votre UDF :

```python showLineNumbers
import fused
fused.secrets["my_secret"]
```

Vous pouvez également lister tous les secrets disponibles avec :

```python showLineNumbers
dir(fused.secrets)
```

## Variables d'environnement

![variables d'environnement](/img/workbench/preferences/env_variables.png)

Les variables d'environnement dans Fused sont préconfigurées par l'équipe Fused. Contactez l'équipe Fused pour en configurer de nouvelles.

Ces variables peuvent être accessibles de la manière habituelle via le module os de Python :

```python showLineNumbers
import os
os.environ["ENV_VAR_NAME"]
```
