Un Canvas est un moyen pour les utilisateurs d'organiser leurs UDFs en différents projets. Cela vous permet d'utiliser Workbench pour des projets complètement différents et non liés et d'organiser vos UDFs de manière plus claire.

Avec Canvas, vous pouvez :
- Enregistrer tous les UDFs ouverts actuels dans un Canvas (après lui avoir donné un nom)
- Télécharger / Télécharger Canvas via Workbench (Cliquez sur les 3 points à côté du nom du Canvas)
- Supprimer Canvas (cela ne supprime pas les UDFs à l'intérieur, seulement le Canvas)
