Le UDF Builder est un IDE pour écrire des [User Defined Functions (UDFs)](/core-concepts/why/) et visualiser leur sortie.

L'analyse de données est intrinsèquement itérative. Elle implique le chargement de données, l'écriture de code, l'exécution du code et la visualisation des résultats. Alors que les développeurs écrivent et déboguent leur code dans le UDF Builder, Fused leur donne un retour immédiat en réexécutant automatiquement l'UDF et en montrant la sortie. Les développeurs peuvent écrire des UDFs dans l'éditeur de code, prévisualiser la sortie sur la carte et styliser la visualisation. Pour aider au débogage, le panneau des résultats affiche les journaux et les erreurs générés par l'UDF.



- Écrire des UDFs dans l'[éditeur de code](/workbench/udf-builder/code-editor/)
- Gérer les UDFs en tant que couches de carte dans le [panneau de navigation](/workbench/udf-builder/navigation/)
- Prévisualiser la sortie de l'UDF sur la [carte](/workbench/udf-builder/map/) et modifier le [stylage de la couche](/workbench/udf-builder/styling/) de sa visualisation
- Voir les journaux de l'UDF dans le panneau [Runtime](/workbench/udf-builder/runtime/)
